Proyecto:

Una red donde diferentes tipos de artistas pueden conectarse para (proximamente) ofrecer sus servicios o solamente hacer publicidad de sus trabajos
La aplicacion de "Salute_csales" iba a estar dedicada a las ventas, pero por temas de tiempo lo deje vacio por ahora.

Funcionalidad:

-El proyecto tiene una funcion de busqueda para filtrar a los artistas por nombre (desde la pagina de cada tipo)
-Tiene una funcion para agregar un artista (desde la pagina de cada tipo


Aplicaciones:

"salute_basic" contiene el menu de inicio, desde donde salen el resto de aplicaciones.

"salute_artist" un menu para cada tipo de artista, con sus respectivas funciones.

"salute_csales" un menu donde los artistas pueden vender sus trabajos u ofrecer sus servicios en forma de comisiones.
